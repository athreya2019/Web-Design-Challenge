Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as it gets closer to the equater. Used OpenWeatherMapAPI to build a weather dataset on over 500 cities for the analysis. Used Matplotlib library to plot various aspects of the weather vs. latitude,factors such as temperature,humidity, cloudiness and wind speed. This site provides data visualizations created as part of the analysis.
